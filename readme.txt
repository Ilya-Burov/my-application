Это Android приложение записная книжка. В нем осуществляется работа на базой данных Sqlite Database, где хранятся сведения о пользователях. Человек, использующий данную записную книжку может создавать , сохранять новых пользователей, редактировать сведения о них . Также с использованием свайпа влево или вправо можно удалить пользователя. Данное приложение пригодится тем, кому необходимо сохранять сведения о различных людях и  быстро c ними работать.
